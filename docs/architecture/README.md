# Legacy Architecture Notes

This folder contains earlier architecture planning notes created during the initial design phase of the Job Market Radar project.

These files may describe intended or future-state models and should not be treated as the current implementation contract.

For the current public documentation, use:

- `docs/architecture.md`
- `docs/data_flow.md`
- `docs/data_catalog.md`
- `docs/local_runbook.md`
- `docs/validation_summary.md`

Known examples of future-state concepts in this legacy folder may include:

- dedicated warehouse skill dictionary models
- dedicated job-skill bridge models
- inactive-job tracking fields planned before the current MVP contract was finalized

The current MVP implementation documents the active state in the root `README.md` and the top-level `docs/*.md` files.
